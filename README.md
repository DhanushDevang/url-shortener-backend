# URL Shortener Service

## Overview
A scalable URL shortening service inspired by Bitly, designed to handle high read traffic with low latency.

## Features (Planned)
- URL shortening
- Fast redirection
- Caching using Redis
- Rate limiting
- Analytics (click tracking)

## Tech Stack
- Java
- Spring Boot
- PostgreSQL (upcoming)
- Redis (upcoming)
- AWS (deployment planned)
